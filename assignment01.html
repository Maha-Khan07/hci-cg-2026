Common HTML Errors and Their Corrections
# Incorrect HTML Why It's wrong Corrected HTML
1 <html> ... <\html> Backslash used in closing tags, HTML always uses forward slash <html> ... </html>
2 <br></br> <br> is self-closing; no closing tag is needed in HTML5. <br>
3 <img src=cat.jpg alt=Cat> Attribute values should always be in quotes. <img src="cat.jpg" alt="Cat">
4 <p><h1>Title</h1></p> Block-level element <h1> cannot be inside <p>. <h1>Title</h1><p>Paragraph text.</p>
5 <ul><li>Item 1<li>Item 2</ul> Missing closing <li> tags; can break rendering. <ul><li>Item 1</li><li>Item 2</li></ul>
